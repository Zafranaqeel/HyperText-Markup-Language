Pengelompokan Elemen HTML

Header, Footer, Main, dan Nav.

Header dan Footer

  • Header

    • Header adalah pengantar atau pembuka konten dalam sebuah elemen <article> atau <section>
    Header biasanya memuat judul dan bar navigasi <nav>
    Elemen yang digunakan: <header> </header>

  • Footer

    • Header adalah catatan kaki dalam sebuah elemen <article> atau <section>
    Header biasanya memuat hak cipta dan dapat menampung link untuk membagikan artikel ke sosial media.
    Elemen yang digunakan: <footer> </footer>


    Catatan: elemen <header> dan <footer> tidak boleh ditulis di dalam elemen <header> dan <footer> lainnya.

Main

Elemen <main> digunakan untuk menampung/mewadahi konten utama dalam <body>
Catatan: <!-- --> merupakan tag komentar pada HTML yang fungsinya untuk menambahkan komentar atau catatan. Teks yang berada dalam tag tersebut tidak akan ditampilkan dalam website.

Nav

Elemen <nav> digunakan untuk menampung sebuah navigasi yang sifatnya penting, contohnya navigasi utama sebuah website.

Sebuah navigation atau navigasi sangat berguna untuk aksesibilitas website karena memungkinkan pengguna mencari konten yang diinginkannya tanpa menelusuri seluruh konten website.

Article, Aside, dan Section

Article

Elemen <article> bertindak sebagai container untuk independent content pada sebuah halaman, artinya konten yang utuh dan tidak terkait dengan konten lain, contohnya adalah artikel, blog, dan konten lainnya.
Jika dalam sebuah website terdapat beberapa artikel, tiap artikel seharusnya berada pada elemen <article>-nya masing-masing.

Aside

Elemen <aside> memiliki dua tujuan, tergantung penempatannya di dalam elemen <article> atau tidak.

  • Ketika elemen <aside> ditempatkan di dalam elemen <article>, elemen ini dapat berisi informasi yang berhubungan dengan konten artikel tersebut.
  • Ketika elemen <aside> ditempatkan di luar elemen <article>, elemen ini dapat berisi informasi yang berhubungan dengan keseluruhan halaman.
Elemen <aside> biasanya berada di samping dari sebuah elemen yang menampungnya.

Section

Elemen yang memiliki kesamaan konten dan memiliki heading dapat dikelompokan dengan elemen <section>

Generic Element

Generic Element adalah elemen umum yang bisa dikustomisasi untuk menggambarkan konten dengan tepat.
Ada dua Generic Element yang dapat dimanfaatkan, yaitu Div dan Span

Div

Elemen <div> merupakan sebuah wadah (container) yang bersifat umum untuk menampung beberapa konten
Elemen ini tidak memberikan efek apapun pada konten sebelum diberikan style menggunakan CSS
Elemen <div> lebih sering digunakan untuk mengelompokkan sebuah konten untuk memudahkan styling dengan menggunakan atribut class atau id

Span

Elemen <span> hampir sama dengan <div>, bedanya elemen ini digunakan sebagai phrase element dan tidak terdapat line breaks ketika menggunakannya.
