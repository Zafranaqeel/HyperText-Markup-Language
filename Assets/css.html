Cascading Style Sheet

Keuntungan ketika menerapkan CSS

  1. Dapat mengontrol dan menerapkan layout (tata letak) secara presisi
  2. Menghindari pekerjaan yang berulang-ulang dalam menerapkan styling
  3. Didukung banyak browser

Cara stylesheet bekerja

  1. Dimulai dari sebuah dokumen yang sudah ditandai dengan tag elemen HTML.
  2. Menuliskan aturan styling untuk menentukan tampilan elemen HTML.
  3. Melampirkan file .css pada HTML.

Menulis Aturan Styling

Terdapat 2 bagian pada sebuah rule. Bagian pertama adalah identitas elemen (selector) dan kedua adalah deklarasi atau instruksi yang akan diterapkan pada sebuah selector.

    h1{
      color: green;    
    }
            
    p{
      font-size: small;
      font-family: sans-serif;
    }

Selector

Pada contoh di atas, <h1> dan <7> digunakan sebagai selector. Properti dan nilainya yang terdapat dalam declaration/declaration block akan diterapkan pada seluruh elemen <h1> dan <p> yang ada pada dokumen HTML.

Declaration

Deklarasi terdiri dari pasangan properti dengan nilainya. Satu rule bisa diberi lebih dari satu deklarasi seperti contoh pada selector p di atas. Setiap deklarasi harus diakhiri dengan semicolon (;) sebagai tanda diakhirinya sebuah deklarasi.
CSS tidak memperhatikan spasi, sama seperti bahasa pemrograman pada umumnya. Walaupun seperti itu, untuk satuan nilai semisal px, em, dan lainnya harus dituliskan tanpa spasi pada nilainya. Contohnya seperti dibawah: 

            h1{
                margin: 2em;
            }

Komentar Pada CSS

Komentar digunakan untuk menjelaskan kode dan dapat membantu ketika mengedit kode di lain waktu. Hal yang ditulis tidak akan ditampilkan di browser namun bisa dilihat pada text editor.
Cara penulisan comment pada CSS adalah sebagai berikut:
/* */

Melampirkan Styling Pada Dokumen HTML

  • External Style Sheet

    • External Style Sheet adalah berkas terpisah yang di dalamnya hanya ada CSS rules. Berkas harus berekstensi .css dan dihubungkan dengan dokumen HTML. Dengan cara ini, satu berkas styling dapaat digunakan oleh banyak berkas HTML.

    Cara Menghubungkan berkas .css dengan dokumen HTML 
        <head>
        <meta charset="8-UTF">
        <title>Styling</title>
        <link href="Style.css">
    </head>
    Catatan: Nama berkas yang mengandung "min.css" adalah penamaan format berkas CSS yang sudah diminimalkan dengan menghilangkan white space yang tidak digunakan.

  • Embedded Style Sheet

    • Embedded Style Sheet adalah kumpulan rules yang dituliskan dalam berkas HTML dengan menggunakan elemen <style>. Dengan demikian, CSS rules yang dituliskan hanya dapat dicakup oleh satu berkas HTML. Penulisan rules harus dituliskan dalam elemen <style>.

    Penulisan Rules 
        <head>
        <title>
        <style>
          h1{
            color: blue;
          }
        </style>
    </head>

  • Inline Style

    • Inline Style adalah stylingyang diterapkan secara langsung pada elemen HTML dengan menggunakan atribut style.
    Dalaam metode ini kita tidak memerlukan selector karena style hanya diterapkan pada elemen tempat atribut style diterapkan.

    Inline Style 
    <p> style="color: blue;"> paragraf </p>

CSS Conception

Inheritance

Dapat mewarisi properti style tertentu dari suatu elemen ke elemen-elemen di dalamnya (child-elements). Contohnya CSS rules yang ditetapkan untuk elemen <body> akan diterapkan pada seluruh child-elements secara otomatis.

Group Selector

Jika beberapa selector yang berbeda memiliki penerapan properti-properti yang sama, kita dapat menggabungkannya menggunakan group selector.

Contoh Penggunaan 
    h1{
        font-family: sans-serif;
    }
    h2{
        font-family: sans-serif;
    }
    /*Diubah menjadi*/
    h1, h2{
        font-family: sans-serif;
    }

Rule Order

cascading artinya "mengalir". Demikian halnya dengan alur kerja CSS dalam membaca kode, mengalir dari atas ke bawah. Oleh karena itu, urutan dalam penulisan rules harus diperhatikan, terutama saat terjadi konflik. Konflik dapat terjadi ketika menerapkan beberapa styling pada satu dokumen HTML dan menimpa styling yang telah diterapkan sebelumnya baik disadari maupun tidak.

HTML CSS 
<!DOCTYPE html>
<html>
    <head>
    <meta charset="UTF-8" />
    <title>Judul</title>

    <link rel="stylesheet" href="Style.css">

    <style>
        p{
         color: blue;
        }
    </style>
    </head>
    <body>
    <p>
      Lorem ipsum dolor sit amet.
    </p>
    </body>
</html>
                
  p{
   color: red;
  }
Dalam contoh diatas, warna yang akan diterapkan pada elemen <p> adalah warna biru karena merupakan warna yang paling akhir didefinisikan.

Namun, kita bisa membuat sebuah property styling agar dianggap penting oleh browser untuk diterapkan dan urutan tidak diperhatikan dengan menambahkan keyword !important pada akhir nilai propertinya.

HTML CSS 
<!DOCTYPE html>
<html>
    <head>
    <meta charset="UTF-8" />
    <title>Judul</title>

    <link rel="stylesheet" href="Style.css">

    <style>
        p{
         color: blue;
        }
    </style>
    </head>
    <body>
    <p>
      Lorem ipsum dolor sit amet.
    </p>
    </body>
</html>
                
  p{
   color: red !important;
  }
Dalam contoh diatas, warna yang akan diterapkan pada elemen <p> adalah warna merah karena karena terdapat keyword !important pada akhir nilai properti.

Selektor Dasar

Type Selector

Menggunakan nama elemen sebagai target untuk diterapkannya rule.

Class Selector

Menetapkan target elemen berdasarkan nilai dari atribut class yang diterapkan pada elemennya. Untuk penulisan selector-nya diawali dengan tanda titik (.) dan dilanjutkan dengan nama class nya. <!DOCTYPE html>

HTML CSS 
    <!DOCTYPE html>
    <html>
    <head>
       <meta charset="UTF-8"
       <title>Judul</title>
    </head>
    <body>
      <p class="baris1">dolor sed viverra</p>
      <p class="baris2">in nibh mauris</p>
      <p class="baris3">a diam sollicitudin</p>
    </body>
    </html>
                    
    .baris1{
        color: red;
    }
    
    .baris2{
        color: blue;
    }

    .baris3{
        background-color: green;
    }
Output (hasil) dari ketiga elemen <p> tersebut masing-masing adalah:
baris 1 merupakan teks berwarna merah
baris 2 merupakan teks berwarna biru
baris 3 merupakan teks dengan latar belakang berwarna hijau

Class bersifat sharable sehingga dapat diterapkan pada banyak elemen dengan tipe elemen yang berbeda-beda seperti pada contoh di bawah: 
    <h1 class="baris1">Heading dengan teks berwarna merah</h1>
    <p class="baris1">Paragraf dengan teks berwarna merah</p>
                    
    <h1 class="baris1">Heading dengan teks berwarna merah</h1>
    <p class="baris1 baris3">Paragraf dengan teks berwarna merah dan background berwarna hijau</p>

ID Selector

Menetapkan target elemen berdasarkan nilai dari atribut id yang diterapkan pada elemennya, mirip dengan class. Kebanyakan atribut ini digunakan untuk memberikan sebuah arti pada generic element seperti <div> dan <span>. Namun, atribut ini tidak bersifat shareable, yang berarti nilai id hanya bisa digunakan pada satu elemen saja.
Gunakan tanda (#) untuk menetapkan selector menggunakan id.

HTML CSS 
    <!DOCTYPE html>
    <html>
    <head>
       <meta charset="UTF-8"
       <title>Judul</title>
    </head>
    <body>
      <div id="greenback">
        <p>
          Div dengan id greenback
        </p>
      </div>
      <div>
        <p>
          Div tanpa id
        </p>
      </div>
    </body>
    </html>
                    
    #greenback{
        background-color: green;
    }
Div dengan id greenback memiliki teks dengan latar belakang berwarna hijau, sedangkan div yang tidak memiliki id tidak berubah karena tidak terkait dengan selector manapun.

Attribute Selector

Merupakan cara menetapkan target elemen berdasarkan sebuah atribut yang digunakan atau bahkan bisa lebih spesifik dengan nilainya.

HTML CSS 
    <!DOCTYPE html>
    <html>
    <head>
       <meta charset="UTF-8"
       <title>Judul</title>
    </head>
    <body>
      <a href="URL">href attribute</a>
    </body>
    </html>
                    
    a[href]{
        color: red;
    }
Syntax Deskripsi
attr Menargetkan elemen yang menerapkan atribut attr.
[attr=value] Menargetkan elemen yang menerapkan atribut attr dengan nilai value
[attr~=value] Menargetkan elemen yang menerapkan atribut attr dan salah satu nilainya adalah value.
[attr^=value] Menargetkan elemen yang menerapkan atribut attr dan nilainya diawali dengan nilai value.
[attr$=value] Menargetkan elemen yang menerapkan atribut attr dan nilainya diakhiri dengan value.
[attr*=value] Menargetkan elemen yang menerapkan atribut attr dan nilainya mengandung value.

Universal Selector

Digunakan untuk diterapkan pada seluruh elemen. Namun, selector ini juga bisa secara spesifik menargetkan sebuah elemen dengan menggabungkannya bersama selector yang lain.

Combinators

Adjacent Sibling Selector

Menggabungkan gua buah basic selector dengan menggunakan tanda (+) diantara keduanya. 

    img + p{
        color:green;
    }
Adjacent Sibling Selector terdiri dari dua buah target elemen, tetapi hanya elemen kedua yang menerapkan rule selama elemen tersebut dituliskan langsung setelah elemen pertama pada berkas HTML. Selain itu, kedua elemen tersebut harus berada dalam induk elemen yang sama.

General Sibling Selector

Mirip seperti Adjacent Sibling Selector, tetapi rules akan diterapkan pada seluruh elemen kedua yang berada setelah elemen pertama selama masih memiliki induk yang sama, walaupun posisi dari elemen kedua tidak berada tepat setelahnya. Menggunaakan simbol tilda (~) untuk menetapkan elemennya. 

    img ~ p{
        color: blue;
    }
Rule di atas akan diterapkan pada elemen paragraf yang berada setelah elemen img selama masih di dalam induk yang sama.

Child Selector

Menggabungkan dua buah basic selector dengan menggunakan tanda greater than (>) diantara basic selectornya. 

    div > p{
        background-color: red;
    }
Pada contoh di atas, CSS rule akan diterapkan pada seluruh elemen paragraf yang berada dalam elemen div secara langsung.

Descendant Selector (space)

Mirip seperti Child Selector, tetapi hierarkinya lebih luas karena rule diterapkan pada seluruh elemen yang menjadi turunannya walaupun secara tidak langsung. Selector ini menggunakan spasi dalam menggabungkan dua basic selector. 

    div p{
        background-color: blue;
    }
Dalam contoh di atas, CSS rule akan diterapkan pada seluruh elemen paragraf yang merupakan turunan dari elemen div.

Pseudo Selector

Pseudo Selector berbeda dari selector yang sudah dibahas sebelumnya. Selector ini menargetkan elemen pada bagian yang "tidak terlihat", seperti sifat pada elemen sehingga untuk menetapkannya tidak bisa menggunakan selector biasa. salah satu contoh yang paling sering diterapkan adalah :hover, pseudo selector tersebut digunakan untuk menetapkan rule ketika curcor diarahkan ke target elemen. 

    a:hover{
        color: blue;
    }
Rule di atas menyebabkan elemen anchor menerapkan warna biru ketika kursor diarahkan pada elemen tersebut.

Pseudo Class Selector

Merupakan sebuah class "semu" yang sebenarnya ada pada tiap elemen HTML.
Untuk menetapkan rule hanya ketika sebuah tautan telah dikunjungi dapat menggunakan (:visited).
Untuk menetapkan rule hanya ketika sebuah elemen diarahkan dengan kursor dapat menggunakan (:hover).
Untuk menggunakan pseudo-class, kita gunakan tanda titik dua (:) pada basic selector dan diikuti dengan pseudo-class nya.

Pseudo Elemen Selector

Merupakan sebuah elemen "semu" yang sebenarnya ada, tetapi tidak tampak secara tertulis. Selector ini bisa digunakan ketika kita menambahkan konten tepat sebelum dan setelah sebuah elemen paragraf. Tidak perlu menuliskan elemen/konten baru pada aberkas HTML untuk menambahkan konten. Namun, kita dapat memanfaatkan pseudo-element untuk melakukannya. Pseudo-element yang dimaksud adalah ::before dan ::after . Konten yang dibuat dengan pseudo-element dituliskan pada CSS. Untuk menggunakan pseudo-element, kita gunakan dua buaha tanda titik dua (::) dan diikuti dengan nama pseudo-element-nya.

Formatting Text

Font Styling

Dalam CSS, font ditentukan menggunakan beberapa paket properti font yang dapat mengatur tipe font, ukuran, ketebalan, dan style. Berikut daftar properti font yang akan dipelajari:

  • font-family: Menetapkan jenis font yang akan diterapkan pada target.
  • font-size: Menentukan ukuran pada teks.
  • font-weight: Menentukan ketebalan pada teks.
  • font-style: Menetapkan styling yang diterapkan pada teks.
  • font-variant: Menentukan teks untuk mengunakan gara small caps(huruf kapital kecil.)
  • font: Sebagai shorthand dari properti font yang ada.

Font-family

    body{
        font-family: sans-serif;
    }
Pada rule tersebut, kita mengubah standar font yang digunakan browser dengan font "sans-serif". Untuk nilai dari properti ini dapat lebih dari satu (font stack), tujuannya adalah sebagai fallback jika terjadi kegagalan dalam menggunakan font yang kita gunakan. 

    p{
        font-family: Arial, "Open Sans", sans-serif;
    }
    
    selector{
        property-name: Value1, Value2, Value3;
    }

Penjelasan:

  • Selurh nilai font yang bukan merupakan generic font harus dituliskan secara kapital. Contohnya "Arial" bukan "arial".
  • Gunakan tanda koma (,) untuk memisahkan antara nilai font yang digunakan.
  • Selalu tanda kutip (") untuk membungkus nilai font yang memiliki spasi pada namanya
Catatan: Pastikan untuk menggunakan generic font families pada akhir nilai properti karena nilai ini dipastikan didukung oleh seluruh browser saat ini. Berikut adalah nilai-nilai generic font families:
Generic Font Families Deskripsi Contoh Font
Serif Jenis font yang memiliki runcing pada garis akhir karakternya. Times New Roman
Sans-serif Jenis font yang tidak meruncing pada garis akhir karakternya. Open Sans, Fira Sans, dan lainnya
Monospace Jenis font yang memiliki nilai lebar tiap karakternya sama. Consolas
Cursive Jenis font yang tampak seperti handwriting atau hasil tulisan tangan.
Fantasy Jenis font yang merepresentasikan karakteristik yang menyenangkan.
System-ui Jika menerapkan nilai ini maka font yang diterapkan akan sama seperti font yang digunakan pada sistem operasi kita.
Math Jenis font yang digunakan untuk penulisan rurmus-rumus matematika.
Emoji Jenis font yang digunakan untuk menampilkan emoji.
Fangsong Jenis font yang menampilkan gaya penulisan Chinese.

@font-face

CSS memiliki fitur yang digunakan utuk memasukkan font eksternal ke dalam CSS, yakni menggunakan @font-face. Berikut adalah contoh penulisannya pada CSS: 

    @font-face{
        font-family: "Nama Font"
        src: url('File Font.TIF')
    }
Setelah mendefinisikan font-face tersebut, ada nilai font-family baru bernilai "Nama Font" untuk diterapkan pada elemen HTML. 
    p{
        font-family: "Nama Font";
    }

Font-size

Untuk mneetapkan ukuran font, kita perlu menerapkan properti font-size. Dengan menetapkan nilai dari properti ini dengan menuliskan langsung nilai dan satuannya. 

    h1{
        font-size: 1.5em;
    }
Hal yang perlu diperhatikan adalah ketika menuliskan nilai dan satuannya. Pastikan tidak ada jarak (spasi) diantaranya karena rule tidak akan diterapkan pada target jika terdapat spasi diantara nilai dan satuan.

Satuan dalam menetapkan ukuran font terbagi dua jenis:

  • Relative Unit

    • Satuan yang nilainya tergantung pada suatu hal. Contohnya ukuran viewport, induk elemen, tau ukuran teks standar.

  • Absolute Unit

    • Satuan yang nilainya telah ditentukan atau doigunakan dalam dunia nyata.

Relative Unit

Satuan Relative to Fungsi
em Font size Satuan relatif terhadap ukuran font yang sedang digunakan pada elemen (contohnya 2em berarti 2 kali lebih besar dari ukuran font seharusnya).
ex Font height Satuan relatif terhadap tinggi font saat ini, satuan ini sangat jarang sekali digunakan.
rem Font size Mirip seperti em, tetapi rem merupakan satuan relatif terhadap ukuran font dari root element.
ch Font width Satuan relatif terhadap lebar dari karakter "0" nol.
vw Viewport width Satuan relatif terhadap 1% lebar viewport. Contoh 1vw=1% dari tinggi viewport. Satuan ini tidak didukung pada browser IE8 ke bawah.
vh Viewport height Satuan relatif terhadap 1% tinggi viewport. Contoh 1vh = 1% dari tinggi viewport. Satuan ini tidak didukung pada browser IE8 ke bawah.

Absolute Unit

Satuan Fungsi
px Menetapkan nilai font berdasarkan ukuran pixel.
pt Menetapkan nilaia font berdasarkan ukuran points (1/72 inch di CSS 2.1).
pc Menetapkan nilai font berdasarkan picas (1 pica = 12 point).
mm Menetapkan nilai font berdasarkan milimeters.
cm Menetapkan nilai font berdasarkan centimeters.
in Menetapkan nilai font berdasarkan inches.

Selain dengan menetapkan nilaai dan satuannya secara langsung, kita juga bisa menggunaskan nilai persentase untuk mengatur ukuran font. 

    body{
        font-size: 16px;
    }

    h1{
        font-size: 150%;
    }
Pada contoh di atas, ukuran font dari element <h1> seharusnya memiliki ukuran 16px karena mewarisi dari induk elemennya (body). Namun, dibawahnya terdapat rule yang menargetkan secara spesifik untuk elemen h1 untuk menerapkan font sebesar 150% dari ukuran induknya. Dampaknya, elemen <h1> akan tampak 50% lebih besar dari elemen yang lain yang ada di dalam body.
Hal yang terakhir, kita juga bisa menentukan ukuran font dengan menuliskan kata kunci secara spesifik yang tersedia pada CSS. Kata kunci tersebut adalah : xx-small, x-small, small, medium, large, x-large, xx-large.
